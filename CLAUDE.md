# kpop-hangul

K-pop 단어로 한글 배우기 앱 (외국인 대상)

## Level: Dynamic

## Tech Stack
- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Zustand (state)
- TanStack Query (data fetching)
- bkend.ai (BaaS - auth, database)

## Project Rules
- 한국어 UI + 영어 번역 병행
- 저작권 없는 콘텐츠만 사용 (단어, 숙어, 노래 제목, YouTube 링크)
- K-pop 가사 직접 인용 금지
